ESLint is designed to lint JavaScript and TypeScript. For other file types — CSS, HTML, XML, GraphQL, and more — this config integrates external formatters via [`eslint-plugin-format`](https://github.com/antfu/eslint-plugin-format).

## Quick start

<Steps>
  <Step title="Enable formatters in your config">
    Pass `formatters: true` to enable all formatters with sensible defaults. When XML and SVG support requires `@prettier/plugin-xml`, those formatters are only enabled if the plugin is already installed.

    ```js eslint.config.mjs
    import antfu from '@antfu/eslint-config'

    export default antfu({
      formatters: true,
    })
    ```

    Or pick specific formatters:

    ```js eslint.config.mjs
    import antfu from '@antfu/eslint-config'

    export default antfu({
      formatters: {
        css: true,
        html: true,
        markdown: 'prettier',
      },
    })
    ```
  </Step>
  <Step title="Install the base dependency">
    <CodeGroup>
      ```bash npm
      npm i -D eslint-plugin-format
      ```
      ```bash pnpm
      pnpm i -D eslint-plugin-format
      ```
      ```bash yarn
      yarn add -D eslint-plugin-format
      ```
    </CodeGroup>
  </Step>
</Steps>

## Formatter options

### CSS

Formats `.css`, `.pcss`, `.postcss`, `.less`, and `.scss` files, and also `<style>` blocks in Vue SFCs. Uses Prettier.

```js eslint.config.mjs
export default antfu({
  formatters: {
    css: true,
  },
})
```

No additional packages are needed beyond `eslint-plugin-format`.

Formats `.html` files using Prettier.

export default antfu({

### Slidev

Enables Prettier formatting for Slidev markdown presentation files (`slides.md` by default). Only works when `markdown` is also enabled with Prettier.

<Steps>
  <Step title="Enable Slidev formatting">
    ```js eslint.config.mjs
    export default antfu({
      formatters: {
        markdown: 'prettier',
        slidev: true,

        // Or target specific files:
        slidev: {
          files: ['**/slides.md'],
        },
      },
    })
    ```
  </Step>
  <Step title="Install the Slidev Prettier plugin">
    <CodeGroup>
      ```bash npm
      npm i -D prettier-plugin-slidev
      ```
      ```bash pnpm
      pnpm i -D prettier-plugin-slidev
      ```
      ```bash yarn
      yarn add -D prettier-plugin-slidev
      ```
    </CodeGroup>
  </Step>
</Steps>

<Warning>
  The `slidev` option only works when `markdown` is set to `true` or `'prettier'`. It will throw an error if markdown is set to `'dprint'` or not enabled.
</Warning>

## Custom Prettier options

Pass `prettierOptions` to override the default Prettier configuration used across all formatters:

```js eslint.config.mjs
export default antfu({
  formatters: {
    css: true,
    html: true,
    prettierOptions: {
      printWidth: 100,
      singleQuote: true,
      semi: false,
    },
  },
})
```

Default Prettier options are derived from the active `stylistic` settings (indent, quotes, semi) and include `printWidth: 120`, `endOfLine: 'auto'`, and `trailingComma: 'all'`.

## Custom dprint options

Pass `dprintOptions` to provide custom dprint configuration for formatters that support it (currently Markdown with `dprint`):

```js eslint.config.mjs
export default antfu({
  formatters: {
    markdown: 'dprint',
    dprintOptions: true,
  },
})
```

dprint options default to matching the active `stylistic` settings for `indentWidth`, `quoteStyle`, and `useTabs`.
