`@antfu/eslint-config` is a personal ESLint config preset by [Anthony Fu](https://github.com/antfu). It uses [ESLint Flat config](https://eslint.org/docs/latest/use/configure/configuration-files-new) and is designed to be the only linting and formatting tool you need — replacing Prettier entirely.

<Warning>
  This is a personal config with strong opinions. Changes might not work for every use case. If you use this config directly, review the changelog each time you update. You can always fork it for more control.
</Warning>

## Style principles

The formatting choices in this config follow a clear philosophy:

- **Minimal for reading** — less visual noise, easier to scan
- **Stable for diff** — formatting decisions reduce noisy diffs in code review
- **Consistent** — one style, applied everywhere

Concretely, this means:

- Single quotes, no semicolons
- Sorted imports and dangling commas
- 2-space indentation
- Formatting via [ESLint Stylistic](https://github.com/eslint-stylistic/eslint-stylistic) (not Prettier)

<Tip>
  Interested in the philosophy behind this config? Anthony gave a talk at JSNation 2024: [ESLint One for All Made Easy](https://gitnation.com/contents/eslint-one-for-all-made-easy).
</Tip>

## Requirements

- ESLint v9.5.0 or higher
- Node.js (current LTS recommended)
