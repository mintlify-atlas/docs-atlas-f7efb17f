`@antfu/eslint-config` is a highly opinionated, composable ESLint flat config preset designed for modern JavaScript and TypeScript projects. It replaces Prettier for formatting, auto-detects your frameworks, and gets you linting with a single line of code.

## Why @antfu/eslint-config?

<CardGroup cols={2}>
  <Card title="One line of config" icon="bolt">

## Quick start

<Steps>
  <Step title="Run the setup wizard">
    The CLI wizard installs dependencies and generates your `eslint.config.mjs` automatically.

    ```bash
    pnpm dlx @antfu/eslint-config@latest
    ```
  </Step>
  <Step title="Add lint scripts">
    Add these scripts to your `package.json`:

    ```json package.json
    {
      "scripts": {
        "lint": "eslint",
        "lint:fix": "eslint --fix"
      }
    }
    ```
  </Step>
  <Step title="Run ESLint">
    ```bash
    pnpm lint
    pnpm lint:fix
    ```
  </Step>
</Steps>

<Tip>
  If you prefer manual setup, see the [Getting Started](/getting-started) guide for step-by-step instructions.
